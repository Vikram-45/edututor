# EduTutor

EduTutor is an educational web application that provides:
- AI-powered chatbot for subject assistance and study guidance
- MCQ quiz module for self-assessment
- Role-based dashboard for students and teachers

## What this project includes

- `Backend/edututor`: Django backend with REST endpoints and authentication (students/teachers/accounts)
- `Frontend/edututor`: React + Vite frontend with Tailwind UI and chatbot interface

## Key features

- Intelligent, iterative Q&A flow powered by backend chatbot endpoint
- User session handling for dashboards and role-specific navigation
- Responsive UI with a modern education platform design

## Goal

Make learning easier with a low-friction interface, quick answers, and a streamlined teacher/student experience. This README highlights project purpose and architecture, not install steps.
